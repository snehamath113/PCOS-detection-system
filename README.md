# PCOS Detection System using Machine Learning

## Description
This project predicts the presence of PCOS using medical parameters
and Machine Learning (Random Forest Classifier).

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train_model.py

3. Run prediction:
   python predict.py
